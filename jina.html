JINA PROJECT

Jina is a MLOps framework that empowers anyone to build multimodal AI services via cloud native technologies. It uplifts a local PoC into a production-ready service. Jina handles the infrastructure complexity, making advanced solution engineering and cloud-native technologies accessible to every developer. I am one of the contributors involved with the developing of this platform. I have learnt a great deal on best practices on developing an ML modal at scale.

Check out JINA github for more information
